# Seguridad IoT

Capacidad transversal que aplica a todas las capas del modelo IoT.

## Capacidades de seguridad genéricas

### En la capa de aplicación
- Autenticación
- Autorización
- Confidencialidad de datos
- Integridad
- Protección de la privacidad
- Auditorías
- Antivirus

### En la capa de red
- Autenticación
- Autorización
- Confidencialidad de señalización y datos
- Protección de integridad

### En la capa de dispositivo
- Autenticación
- Autorización
- Validación de integridad del dispositivo
- Control de acceso
- Confidencialidad de datos

## Capacidades de seguridad específicas
- Relacionadas con requisitos particulares del servicio
- Ejemplo: pagos móviles
