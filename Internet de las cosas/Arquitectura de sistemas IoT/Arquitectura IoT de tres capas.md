# Arquitectura IoT de tres capas

Modelo conceptual compuesto por tres capas principales.

## Capa de percepción
- Responsable de la recolección de información.
- Utiliza sensores físicos y componentes de objetos inteligentes.
- Transmite la información procesada a la capa de red mediante interfaces de servicio.

## Capa de red
- También conocida como capa de transmisión.
- Acepta los datos provenientes de la capa de percepción.
- Reenvía los datos a las interfaces de aplicación.
- Utiliza redes integradas, Internet y otras tecnologías de comunicación.

## Capa de aplicación
- Provee los servicios solicitados por los usuarios.
- Puede ofrecer servicios de dAPP como:
  - Almacenamiento de datos
  - Big Data
  - Minería de datos
  - Análisis semántico
