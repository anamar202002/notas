# Arquitecturas IoT propuestas

En la literatura se han propuesto múltiples arquitecturas para sistemas IoT.

Estas arquitecturas difieren:
- En funcionalidades
- En terminología técnica
- En nivel de abstracción

Esta diversidad limita la interoperabilidad entre sistemas IoT, ya que aún no existe una arquitectura de referencia única adoptada de manera generalizada.
