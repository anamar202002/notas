# Capa de apoyo a aplicaciones y servicios IoT

También conocida como capa de datos.

## Capacidades de soporte genéricas
- Recolección de datos
- Limpieza de datos
- Almacenamiento
- Procesamiento

## Capacidades de soporte específicas
- Funcionalidades adaptadas a necesidades particulares
- Ejemplo: semántica de datos
