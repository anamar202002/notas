# Capa de red IoT

Responsable de la transmisión segura de datos desde los sensores físicos hacia los sistemas de procesamiento.

## Capacidades
### Capacidad de red
- Control de conectividad
- Control de acceso
- Gestión de movilidad
- Autenticación, autorización y contabilidad

### Capacidad de transporte
- Transporte de datos de servicios y aplicaciones IoT
- Transporte de información de control y gestión
