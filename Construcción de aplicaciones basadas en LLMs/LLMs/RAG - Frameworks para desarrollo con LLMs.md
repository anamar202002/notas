### Principales frameworks

- [[LangChain]]: orquestación, prompts, integración con herramientas.
    
- **LlamaIndex**: indexación y recuperación de datos, ideal para RAG.
    

### Capacidades clave

- Integración con documentos, APIs, bases de datos y servicios cloud.
    
- Clientes unificados para múltiples proveedores de modelos.
    
- Gestión y versionado de prompts.
    
- Pipelines reutilizables.
    
- Conectores listos para bases vectoriales.
